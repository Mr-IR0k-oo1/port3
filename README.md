# MotionCraft Portfolio

A premium portfolio website for a video editor and motion graphics designer, featuring modern animations, dark/light theme support, and responsive design.

## Features

- **Modern Design**: Clean, professional layout with orange accent colors
- **Smooth Animations**: GSAP-powered animations and scroll effects
- **Dark/Light Theme**: Toggle between dark and light modes
- **Responsive**: Fully responsive design that works on all devices
- **Interactive Elements**: Custom cursor, hover effects, and micro-interactions
- **Portfolio Showcase**: Filterable portfolio grid with project categories
- **Contact Form**: Functional contact form with validation
- **Performance Optimized**: Optimized animations and lazy loading

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern CSS with custom properties and animations
- **JavaScript**: Vanilla JS with ES6+ features
- **GSAP**: Professional animation library
- **Font Awesome**: Icon library
- **Google Fonts**: Inter and Playfair Display typography

## Sections

1. **Hero**: Eye-catching introduction with call-to-action
2. **Clients**: Trusted industry partners
3. **About**: Professional background and expertise
4. **Portfolio**: Featured projects with filtering
5. **Services**: Detailed service offerings
6. **Testimonials**: Client testimonials
7. **Contact**: Contact form and information
8. **Footer**: Additional links and information

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. No build process required - it's a static website

## Deployment

This site is designed to be deployed on GitHub Pages. The workflow is automatically configured in `.github/workflows/deploy.yml`.

## Customization

- Edit the content directly in the HTML files
- Modify colors and styles in `styles.css`
- Update animations in `script.js`
- Replace placeholder content with your actual projects and information

## Browser Support

- Chrome/Chromium (recommended)
- Firefox
- Safari
- Edge
